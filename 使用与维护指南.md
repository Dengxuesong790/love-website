# 💝 使用与维护指南

## 📖 快速开始

### 第一步：查看项目结构
```
LoveByIdea/
├── index.html              # 主页面文件
├── css/                    # 样式文件夹
│   ├── style.css          # 全局样式
│   └── pages.css          # 页面样式
├── js/                     # 脚本文件夹
│   └── main.js            # 主要逻辑
├── images/                 # 图片文件夹
│   ├── photo1.jpg         # 相册照片 1
│   ├── photo2.jpg         # 相册照片 2
│   ├── photo3.jpg         # 相册照片 3
│   ├── photo4.jpg         # 相册照片 4
│   ├── photo5.jpg         # 相册照片 5
│   └── couple-photo.jpg   # 结尾合照
├── music/                  # 音乐文件夹
│   └── background.mp3     # 背景音乐
├── README.md              # 项目说明
├── 部署指南.md             # 部署教程
└── 使用与维护指南.md       # 本文档
```

### 第二步：替换素材
1. **替换照片**（必须）
   - 打开 `images/` 文件夹
   - 按照《图片替换说明.txt》的指引替换照片
   - 建议准备 5 张合照 + 1 张结尾大图

2. **添加音乐**（可选）
   - 打开 `music/` 文件夹
   - 按照《音乐替换说明.txt》的指引添加音乐
   - 如果没有音乐，网站功能不受影响

3. **修改文字**（推荐）
   - 用文本编辑器打开 `index.html`
   - 搜索需要修改的文字内容
   - 替换为你们的故事和告白信

### 第三步：本地测试
- 直接双击 `index.html` 打开
- 或使用浏览器打开该文件
- 测试所有功能是否正常

### 第四步：部署上线
- 参考《部署指南.md》选择部署平台
- 推荐使用 GitHub Pages（免费、简单）
- 部署后获得可分享的链接

## 🔧 日常维护

### 更新照片
1. 准备新照片
2. 重命名为对应的文件名
3. 覆盖 `images/` 文件夹中的旧照片
4. 刷新网页查看效果

### 更新告白信
1. 用文本编辑器打开 `index.html`
2. 找到第 178-210 行的告白信内容
3. 修改为你想说的话
4. 保存并刷新网页

### 更新时间线事件
1. 打开 `index.html`
2. 找到第 67-102 行
3. 修改日期、标题、描述等信息
4. 可以复制整个 `timeline-item` 块来添加更多事件

### 修改计时器起始日期
1. 打开 `js/main.js`
2. 找到第 11 行
3. 修改 `startDate` 的日期
4. 保存即可

## 🎨 自定义修改

### 修改颜色主题
编辑 `css/style.css` 第 9-18 行的 CSS 变量：

```css
:root {
    --primary-pink: #FFB6C1;    /* 主色调 */
    --light-pink: #FFD1DC;      /* 浅粉色 */
    --dark-pink: #FF69B4;       /* 深粉色 */
    --soft-pink: #FFE4E1;       /* 柔粉色 */
}
```

可以改成对方喜欢的颜色组合。

### 修改字体大小
在 `css/pages.css` 中搜索并修改：
- `.couple-names` - 昵称大小
- `.main-title` - 主标题大小
- `.time-number` - 计时数字大小

### 调整动画速度
在 CSS 文件中搜索 `animation` 或 `transition` 属性，修改时间值（如 0.3s 改为 0.5s）。

## 🐛 常见问题排查

### 问题 1：照片不显示
**原因：**
- 文件路径错误
- 文件名不匹配
- 图片格式不支持

**解决方法：**
1. 检查图片是否在 `images/` 文件夹
2. 确认文件名完全一致（包括大小写）
3. 使用 JPG 或 PNG 格式
4. 按 Ctrl+F5 强制刷新浏览器

---

### 问题 2：音乐无法播放
**原因：**
- 现代浏览器禁止自动播放
- 文件路径错误
- 格式不支持

**解决方法：**
1. 点击页面右下角的音乐按钮
2. 确保音乐文件在 `music/` 文件夹
3. 使用 MP3 格式
4. 检查文件名是否为 `background.mp3`

---

### 问题 3：计时器显示异常
**原因：**
- 日期格式错误
- 还没到设定的开始日期

**解决方法：**
1. 打开 `js/main.js`
2. 检查第 11 行的日期格式
3. 确保格式为 `'YYYY-MM-DDT00:00:00'`

---

### 问题 4：滚动动画不流畅
**原因：**
- 浏览器性能限制
- 图片太大
- 开启了太多程序

**解决方法：**
1. 压缩图片大小
2. 关闭其他标签页
3. 升级浏览器到最新版
4. 清理解放内存

---

### 问题 5：部署后无法访问
**原因：**
- DNS 未生效
- 部署配置错误
- 文件上传不完整

**解决方法：**
1. 等待几分钟到几小时
2. 检查部署平台的配置
3. 确认所有文件都已上传
4. 查看部署平台的错误日志

## 💡 高级技巧

### 添加更多照片
1. 在 `index.html` 找到相册部分（约第 117 行）
2. 复制 `<div class="gallery-item">...</div>` 整块
3. 粘贴到合适位置
4. 修改图片路径
5. 照片数量会自动被识别

### 添加更多时间线事件
1. 在 `index.html` 找到时间线部分（约第 67 行）
2. 复制整个 `timeline-item` 块
3. 粘贴到合适位置
4. 修改日期、标题、描述等内容

### 添加统计代码
1. 申请百度统计或 Google Analytics
2. 获取统计代码
3. 粘贴到 `index.html` 的 `<head>` 标签内
4. 保存即可

### 添加favicon 图标
1. 准备一个 32x32 的图标文件
2. 命名为 `favicon.ico`
3. 放到网站根目录
4. 在 `<head>` 中添加：
   ```html
   <link rel="icon" href="favicon.ico">
   ```
