# 💕 王斐和邓雪松 - 爱情纪念网站

这是一个专为爱人【王斐】创建的纯前端情侣爱意专属告白纪念网站，采用温柔治愈、元气可爱、简约高级的混合风格设计。

## 🌸 项目特色

### 功能模块

1. **首页全屏告白封面**
   - 顶部显示专属昵称"王斐和邓雪松"
   - 大标题"我的爱意，都藏在这里"
   - 实时计时器：从 2026 年 4 月 11 日告白日开始精确计算
   - 页面加载渐显动画
   - 右下角背景音乐播放器（可控制开关）

2. **我们的故事时间线**
   - 纵向时间轴布局
   - 重要时间节点：
     * 相识：2026 年 3 月 8 日
     * 第一次约会：2026 年 3 月 8 日
     * 告白：2026 年 4 月 11 日

   - 滚动动画效果

3. **专属相册**
   - 图片无缝轮播展示
   - 点击放大查看功能
   - 支持左右切换和关闭
   - 图片懒加载优化
   - 清晰的图片替换入口

4. **告白信专属页面**
   - 优雅的信纸排版设计
   - 舒适的行间距和字体大小
   - 完整的告白长信内容

5. **结尾页**
   - 情侣合照占位图（可替换）
   - 专属承诺文字："你在身边，在你身边，岁岁年年"
   - 温馨的页面过渡效果

## 🚀 使用说明

### 1. 替换图片

在 `images/` 文件夹中替换以下图片：

- **相册照片**：`photo1.jpg` ~ `photo5.jpg`（共 5 张）
- **结尾合照**：`couple-photo.jpg`

### 2. 添加背景音乐

将您选择的背景音乐文件命名为 `background.mp3` 并放入 `music/` 文件夹。

### 3. 修改告白信内容

编辑 `index.html` 文件中第 178-210 行的告白信内容，替换为您想说的话。

### 4. 调整时间节点

如需修改时间线上的重要日期，编辑 `index.html` 文件中第 67-102 行的时间线数据。

### 5. 修改计时器起始日期

如需修改在一起的计算起始日期，编辑 `js/main.js` 文件第 11 行：

```javascript
const startDate = new Date('2026-04-11T00:00:00');
```

## 🎯 自定义建议

### 修改颜色主题

编辑 `css/style.css` 文件中的 CSS 变量：

```css
:root {
    --primary-pink: #FFB6C1;    /* 主色调 */
    --light-pink: #FFD1DC;      /* 浅粉色 */
    --dark-pink: #FF69B4;       /* 深粉色 */
    --soft-pink: #FFE4E1;       /* 柔粉色 */
}
```

### 添加更多照片

1. 在 HTML 的相册部分添加新的 `.gallery-item`
2. 确保图片已添加到 `images/` 文件夹
3. 相册会自动识别新增的照片数量

### 修改专属昵称

编辑 `index.html` 文件中第 33 行的 `<h1 class="couple-names">` 内容。
